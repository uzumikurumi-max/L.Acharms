# L.A Charms - Bracelet Customizer Placement Guide

## Description
This is a simple **bracelet customizer placement guide**.  
It shows where to place plain charms and allows a preview of a **customized photo** charm.  
It also calculates the **total price** based on the number of charms and customized photos.

**Purpose:**  
- Visual guide for placing charms on the bracelet  
- Preview of customized photos  
- Automatic price calculation  

---

## How to Use
1. Open the website (`index.html`) in your browser.  
2. In the input fields:  
   - Enter the number of **plain charms** you want.  
   - Enter the number of **customized photo charms** you want.  
3. Click **"Show Customized Preview & Price"** button.  
4. The bracelet will display:  
   - The **customized photo** (if any)  
   - The **total price**  

---

## Price List
- **Base bracelet**: included in all orders  
- **Plain charms**: ₱3.88 per charm  
- **Customized photo charms**: ₱25 per charm  

**Example:**  
- 1 plain charm + 1 customized photo → ₱28.88  
- 3 plain charms + 2 customized photos → ₱61.64
